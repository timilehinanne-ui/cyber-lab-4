# Cyber Security Lab 4
## System Statistics Script

This project contains a Bash script that utilizes functions to display system uptime and load averages.

### How to run:
1. Open your Kali Linux terminal.
2. Give the script permission: `chmod +x first_function.sh`
3. Run the script: `./first_function.sh`
